解散派对 026 低成本维护的技艺

熊本张 BenBear42
2021-06-05 (Updated: 2021-06-07)
volunteer

intro

锤子也是工具。并不只是app算作工具。网站/书/厨具，都算工具。本次节目邀请 * 位嘉宾，分享他们体验过的美好的工具。

01.丰元

丰元近一个月内使用过，觉得好用的工具。

milanote。

这是一个可以添加书籍卡片和讨论的视觉化笔记应用。它最大的特点是，无限白板。

1.1 milanote 特点

比如丰元主导的 全球概览 项目，是利用 milanote 的一个很好的示范。项目通过
2个步骤构建：1.素材卡片累积2.卡片之间不断关联。作为白板协作创作的绝佳工具，milanote还可以导出内容和生成语音笔记。

1.2 构建创作的方式

该工具解决的核心问题是，如何构建自我工作空间？ 这个问题伴随着所有的创作者们生息繁衍。比如，全球概览的创始人曾经在长方形集装箱里，做一个很长很长的白板，把所有的素材不断贴上去，以激发创作。

丰元认为这是款被低估的产品，愿意付费购买。但因为其价格幅度，他最终选择了10人一起团购的方式，以获得该工具的美好体验。

2.Gumroad

Gumroad 的主旨是，帮助创作者更容易获得酬劳。创作者可以直接在此平台上进行售卖。比起单纯的工作平台，gumroad集合了更多工具链。

2.1 Gumroad 融资方式

该公司有趣的故事来自其创始人。这位创始人是个奶爸，也曾投资过 clubhouse.目前，它十分热门，源自它的最新的融资模式，直接向公众募资不需要审核。其始因来自，美国最新法律批准公司最高可向民众直接融资500万美元。只需要签署简单的投资协议，个人最高可投资1000美元。

2.2 创作者盈利的可能性

丰元参与该公司投资，刷信用卡送上800美元入股。但对于创作如何生钱？丰元认为，虽然创作本身不赚钱，但其衍生品，比如思路，想法，有可能帮你赚到一笔收入。比如，在经济学书籍中文版出版前，自己翻译了英文，就能提前半年获知信息。这个信息可能就是你赚钱的灵感。

3.Dear Data

人和数据的关系是什么？用数据换取权力？用数据存储隐私？Dear Data 的设计师把数据变成情感和个性的表达。他们将生活里的微小片段，用数据画了出来，比如自己的心情。Dear Data 类似西方世界手账，可以从中观察到发达国家生活得到保障后，在做的自我表达。@jsongal 对 Dear Data 形成的自己的笔记。

02.可爱狼

当一个临时的活动发生在一个平台上，活动和平台共同构成了一个容器。
如何长期维护这个容器？尤其是，如何低成本长期维护此容器？
丰元说，“创作足够久时，还活着的媒介，会吃掉死掉的媒介。”
@可爱狼实践了此种维护：用80块的主播声卡，接到海外应用上，扛起来电台，承载起一个社群和一档24小时节目。
