「莊子晨讀」02:全汝形抱汝生

全汝形抱汝生即，學習珍惜當下。
一方面，多記錄當下的狀態。
從熱愛和信心的角度，加強對當下的感知。
當然也要小心，過度記錄導致焦慮、自卑。
另一方面，閱讀書籍，感受才智之士對當下的態度、作為。

零、念起

你总在幻想五年、七年后的机会。
你总在谋划五年、七年后的事业。
但你却难以谋划一个月内的生涯。

一、「庄子」曰: 耳聾眼瞎

道家 -> 庄子 -> 杂篇 -> 庚桑楚 -> 3

「目之与形，吾不知其异也，盲者不能自见」

自己翻译: 眼睛看到的东西，我却不能辨别出异样，这就像盲人看不见自己一样。

自己评述: 这里加入盲者，是用来形容自己像盲人一样，看不见东西。

「心之与形，吾不知其异也，而狂者不能自得。」

自己翻译: 心灵感受到的东西，我也不能辨别出异样，这就像狂妄者不能感受到他们的心灵一样。

自己评述: 狂妄者大多缺少自知之明，缺少自知是因为其忽略感受，没有在感受自己的心灵上下功夫。

二、「庄子」評析: 兩個矛盾

這裡的情境是: 莊子教授弟子，即便殘缺也能保全志業之道，但其弟子仍不能領悟。遂向莊子描述弟子自己的症狀「目不能見，耳不能聞，心不能感」，簡單的話說便是，「耳聾眼瞎」。

剛開始，莊子給弟子的教誨是「全汝形，抱汝生，無使汝思慮營營。若此三年，則可以幾此言矣。」這話翻譯的意思是，「保全你的身體本來的樣子，擁抱你當下的生活，不要過度思慮。這樣保持三年，就可以達到你想要的境界了。」

然而这里存在两个矛盾。也就是这两个矛盾造成了弟子的困扰。这两个矛盾是:

  1. 「全汝形」 的「形」，「形」的範圍指哪些？是身體？是心靈？還是身、心都包含在內？
  2. 「抱汝生」 ，是否可能？倘若當下的生活令人不甚滿意，那該如何擁抱？

三、「庄子」我見:

  • 「全汝形」

    • 暫且將 「全汝形」的「形」 限定為 「身體」。那麼 「全汝形」 便相當於教導人們要「身體第一」。 平安健康才是王道。
    • 如果研讀歷史，便知，當年冷戰為了所謂的理想，而禁錮一生平安幸福的「有志之士」大有人在。不過當然，受困于安樂生活而碌碌無為的君子，也不在少數。
    • 關鍵還是要看自己更偏愛哪種生活。比如，本人肯定更加偏愛「平安健康」。本人就是能從平凡的生活汲取快樂的小母雞。絕對不是追逐太陽的擎天大雁。

  • 「抱汝生」

    • 市井流傳，當「知行不一」時，還是改變「知」，比較節省成本。 此街頭智慧應有相當的道理。如何擁抱當下的生活？即使當下生活滿目瘡痍？本人也打算採取街頭玩法，改變認知。
    • 也許當下生活自有可愛之處？也許會有更糟糕的生活？
    • 老話說，調整心態、知足常樂 就是簡潔優美的人生算法。

  • 解決念起

    • 回到文初剛開始的雜念 「思慮遠大，卻無法立足當下」。 問題出在哪了呢？
    • 第一，肯定最近身體上過得不錯，能夠 全汝形，才有力氣想七想八。
    • 第二，肯定沒有擁抱生活，不能夠 抱汝生，才總想往遠處使勁。

  • 針對其的方案有什麼呢？

    • 第一，身體上要進行操練，加強運動、勞動，將多餘的精力排洩出去。
    • 第二，態度上要珍惜當下。一方面，多做文章、自傳等，或者設計問題列表，更加精確得記錄當下的狀態。從熱愛和信心的角度，加強對當下的感知。當然要小心，過度記錄導致焦慮、自卑。 另一方面，閱讀書籍，感受才智之士對當下的態度、作為。
