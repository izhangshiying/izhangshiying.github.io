「庄子」晨读01:控制游戏

除了阅读圣人典籍，追求解脱，
也应该有一套原创的安抚你自己的假想控制游戏。

零、「庄子」晨读

  • 「庄子」 -> 杂篇 -> 庚桑楚 -> 3

提问:残缺之人如何能保持圆满？

全汝形

抱汝生

无使汝思虑营营

保全你的身体

贴近你的生活

并且不要过度思虑

一、控制游戏

「庄子」安抚人心胜于「古诗源」。
「古诗源」描述哀伤、挫败，并不传授解决之道。
「庄子」描述难题同时，更多传授了解决之道。
控制感，即便是假想出来的，也能够迅速安抚人心。

也许，你除了阅读圣人典籍，追求解脱，
也应该有一套原创的安抚你自己的假想控制游戏。
你现在手中的控制游戏最常用的是，清扫屋苑和煮饭做菜。
但这经常让你劳累，且耗费时间。
你可以尝试发掘稍轻松、稍快捷，
但又十分贴近现下困境的 控制游戏。

二、当下困境

这自然而然引出了一个问题：
你目前的困境是什么？
是发展一生的志业吗？
是需要为生存发愁吗？
是家庭关系不和谐吗？
这些都不是你目前的困境。
你目前的困境是：

  1. 找不到工作，耽误了社会化练习。
  2. 找不到工作的并发症:抑郁、难以建立节奏感、挫败感。
  3. 找不到合适的工作，挫败感持续积累。

那么，进行与找不到工作相关的控制游戏，
应该是对你而言最为合适了。
