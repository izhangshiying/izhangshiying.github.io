情绪与执行

快乐似乎并不是容易控制的情绪。

今日在图书馆睡了一天，没干什么正事，还做了一个美梦。
昨晚的欢乐情绪，反而造成今日低效的一天。

细心想来，在心情不錯的情況下，自己會進入紅色通道，導致第二天效率下降。
而在心情苦悶時，自己會進入藍色通道，第二天效率反而上升。
如下图循环。

自己仿佛一直在悲喜交加中度过。
快乐似乎并不是容易控制的情绪。
而悲伤消耗心神又过于残酷。

回忆起小学五年级时，站在教室天台看大雨的心情，平静而多情。
那时的我，总能飞去另外一个更加广阔的世界里（文学，尤其是「儿童文学」这本杂志）。

自己想要获得持续、稳定的平静情绪。
不以物喜不以己悲，像契诃夫那般，在熟悉的情绪里，给世界留下一些痕迹。

引用自 [书籍] 可爱的契诃夫 契诃夫 / 童道明 / 商务印书馆 / 2015 / 第167页

你告诉妈妈，不管狗和茶炊怎么闹腾，夏天过后还会有冬天，青春过后还会有衰老，幸福后面跟着不幸，或者相反。人不可能一辈子都健康和快乐，总有什么不幸的事情在等着他。…把一切所发生的都看成是不可避免的，不管这是多么令人伤心。需要做的是，根据自己的力量，完成自己的使命——其他的不用去操心。
