家庭休息技巧

零、缘起

讽刺的是，昨晚刚写完「快乐周日」文章，
宣称自己如何快乐、丰富、轻松，
灾祸就来临了。接着又因为思虑灾祸，失眠了。

大张伟也体会到，失眠的部分原因在于，
刺激过于丰富。
在节俭、限制的环境里，人能更快更好得休息。

工作环境 -> 丰富

家庭环境 -> 极简

家庭环境(物理、精神)是争夺休息的战场。
工作环境(物理、精神)是争夺猎物的战场。
两者是完全不同的专业领域，需要不同的专业技巧。

一、目标

  • 将假日的主要行为限制，休息；
  • 能够快速、准时入睡。

二、家庭休息技巧资源

  • 杨志华(简叔)书籍 -> 不想购买
  • LIVE immediately -> 与主题不够相关
  • I’m so glad you’re here! - Live Simply -> 主要介绍清洁，与休息无关
