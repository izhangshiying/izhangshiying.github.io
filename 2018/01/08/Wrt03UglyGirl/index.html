Wrt03巫山女

巫山女

杜甫有诗「负薪行」，讲述了巫山地区女性由于连年战事男子稀少而导致适龄女性很难出嫁的困境。其诗如下方。

Copy Code

作者：杜甫

夔州处女发半华，四十五十无夫家。
更遭丧乱嫁不售，一生抱恨堪咨嗟。
土风坐男使女立，应当门户女出入。
十犹八九负薪归，卖薪得钱应供给。
至老双鬟只垂颈，野花山叶银钗并。
筋力登危集市门，死生射利兼盐井。
面妆首饰杂啼痕，地褊衣寒困石根。
若道巫山女粗丑，何得此有昭君村。

由于缺乏劳动力，妇女成为主要劳动力，而担负其卖出柴火赡养家庭的重担。在日复一日劳作下，女子的体态都已变形，更加难以选择夫婿。杜甫感叹这些不能出嫁的女子为，巫山女。且一针见血得指出，这场悲剧并不是因为这些女子出生丑陋或性格粗鄙造成的，而是战乱造成。巫山女子成为战乱的牺牲品。然而，战乱频繁发生，平顺偶然出现。如何避免巫山女悲剧反复重演？在综合积极心理学书籍与友人聊天体会，我试图作出回答。图1为积极心理学(参考文献1)提供图片。图2为我与友人闲聊后修改图片。着重说明图2。

图2

  • 横轴:格局，向右延伸为大格局。
  • 纵轴:落地，向上延伸为容易落地。
  • A点:大格局难落地，焦虑状态
  • B点:大格局易落地，兴奋状态
  • C点:无格局易落地，轻松状态
  • O点:中格局中落地，可忍受状态
  • l1:无目的行为方式
  • l2:有目的行为方式

落地l2幸福l1

如何避免巫山女悲剧？需要2个阶段的努力。

阶段1: 沿着l2线 ，从A点到C点，从焦虑到轻松。难落地的大想法最想让头疼。 将格局缩小，将落地程度提高。比如从大的改动缩减到小的改动，从力求优秀的作品缩减到力求平均水准的作品。阶段1方法可总结为附表1(见文末)。

阶段2:离开l2线，向着l1线，从C点到B点，从轻松到兴奋。此阶段较为艰难，因为需要从l2线的固有模式改变到l1线的新模式。l1与l2的本质不同是，有无目的性。l1在积极心理学中，被定义为无目的状态。l2在积极心理学中，被定义为有目的状态(参考文献1)。

踏实ACB与梦幻AOB

从困苦的命运超脱的过程可以被简化为图2的从A点移动到B点。上文介绍的一种路径是A->C->B。还有另外一种路径为A->O->B。A->O代表的是，大大降低落地难度的同时稍微降低格局，实现方法可以是空间的转换，现实案例为游学、搬家、换工作、换城市等。O->B代表的是，在新环境、空间下，顺着新环境的节奏，自然又再次提高了格局，且随着技能日益成熟，行动也越容易落地，其现实案例为会议室里高效讨论、图书馆里勤奋写作、博物馆里艺术鉴赏能力提高等。AOB路径所耗费时间较短，且成效较为明显，为不少人所推崇。但此路径有诸多限制，比如需要较大的物质资源、可能引发生活中的波动，甚至需要梦幻般的机缘。相比之下，ACB路径虽然漫长，但因其引起波动较小、耗费物质资源较少，性价比是不错的。
