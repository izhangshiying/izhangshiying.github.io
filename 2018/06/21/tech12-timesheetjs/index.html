html文件如何引用css格式？

资源

  • timesheet.js/README.md at master · sbstjn/timesheet.js
  • bower/bower: A package manager for the web
  • js和css代码嵌入html文件的方式总结 - CSDN博客
  • CSS in JS 简介 - 阮一峰的网络日志

第一次尝试

  • 安装bower
  • 命令行$ npm install -g bower
  • 返回/usr/local/bin/bower -> /usr/local/lib/node_modules/bower/bin/bower /usr/local/lib └── bower@1.8.4
  • 命令行bower install https://github.com/sbstjn/timesheet.js.git
  • 返回timesheet.js#1354b05750 bower_components/timesheet.js
  • HOW TO将js和css代码嵌入html文件

第二次尝试

  • 问题
    • 自己编辑的html文件无法引入css格式
    • 在浏览器打开html文件后，无法显示timesheet.js格式
  • 步骤
    • 在同一目录(/Users/zsy/Documents/timesheet.js/bower_components/timesheet.js/)下重新安装timesheet.js和编写html文件
    • html文件如下
