「庄子」晨读03:内韄外韄

零、资源

  • 庄子 : 杂篇 : 庚桑楚 : 5 : 中国哲学书电子化计划

原文:夫外韄者不可繁而捉，将内揵；

内韄者不可缪而捉，将外揵。

外、内韄者，道德不能持，

而况放道而行者乎！

一、生僻字

  • 韄 huò

    • 缚在佩刀上的绳子。比喻，束缚。

  • 揵 qián

    • 用肩扛。举，扬：“揵鳍掉尾，振鳞奋翼。”；竖立：“揵六枳而为篱兮。”

  • 揵 jiàn

    • 连接：“淮阳包陈以南揵之江。”；古同“楗”，堵塞河堤决口所用的竹木等材料：“塞瓠子决河，……而下淇园之竹以为揵。”

  • 缪 móu Miào miù

    • miù<名>十捆麻。《说文·糸部》：“缪，枲之十絜也。”
    • móu<形>缠绵，见“绸缪”。
    • Miào姓。《廉颇蔺相如列传》：“为赵宦者令缪贤舍人。”

二、自译

佩刀刀鞘上的绳子应该尽量少而简单。倘若绑刀鞘的绳子有很多，那么绳子将相互缠绕，无法有效使用。

同理，绑柴火的麻绳却应该繁多而扎实。倘若只用一条绳子绑柴火，柴火很容易散掉，造成麻烦。

这两种力量(简单的力量和复杂的力量)，即使是得道高人也难以掌握，何况你这个才入门几日的学徒！

三、书译

外韄:能知 -> 所知
内韄:所知 -> 能知

四、感想——好的比喻

好的比喻虽然直击人心，
却担心解释力太强，
只是线性关系。

比如，在”伊甸园微信群”中，
群友提出「盲目的钟表匠」比喻。
认为，演化论解释力太强了，又没办法 证伪，是个大问题。

三个钟表匠的故事……

有神论者 ：钟表匠和上帝（论证人的精致结构来自上帝）

道金斯：盲眼钟表匠（论证演化无需上帝）

西蒙：钟表匠演化逻辑分层结构。（论证演化的复杂性事物，多是层级结构）

那么，好的比喻可以怎样使用呢？
就像如何面对 优美的画面 呢？
大概也就是 放松身心吧。
