下载github远程库中的specific branch

我是小熊🐻
2018-08-10

0.缘由

  • 需要在协同仓库(远程)中创建1个自己主管的新分支，并在本地作修改，推送到远程新分支
  • 整个协同仓库有56个branch
  • 整个协同仓库大小接近500m
  • 目标
    • 本地只与协同仓库(远程)自己主管的那1个新分支建立联系

1.资源

  • How to clone a specific Git branch? - Stack Overflow

2.命令行

mkdir $LOCALFOLDER

cd $LOCALFOLDER

git init

git remote add -t $BRANCH -f origin $REMOTE_REPO

git pull origin $BRANCH_REMOTE:$BRANCH_LOCAL

3.结果

  • 本地master分支成功与远程库zsy分支建立联系。
  • 且只git clone了zsy分支内容到本地目录中。
